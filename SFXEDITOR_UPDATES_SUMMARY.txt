5. QUICK REFERENCE
================================================================================

To create a new SFX:
  1. Open SFX Editor
  2. Choose preset or create new
  3. Adjust sliders
  4. Click Play to test
  5. Save as assets/sfx/name.vsfx
  6. Use in code: PLAY_SFX("name")

Parameter Ranges Quick:
  frequency:      55-1760 Hz
  duration_ms:    20-2000 ms
  attack/decay:   0-500 ms
  sustain:        0-15 (volume)
  release:        0-1000 ms
  peak:           1-15 (volume)
  pitch mult:     0.1-4.0x
  noise period:   0-31
  arpeggio speed: 10-200 ms

Common Chords (semitones):
  Major:          [0, 4, 7]
  Minor:          [0, 3, 7]
  Octave:         [0, 12]
  5th:            [0, 7]
