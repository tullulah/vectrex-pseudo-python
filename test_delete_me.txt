# Archivo de prueba para eliminar

Este es un archivo de prueba que se puede eliminar desde el TreeView.

Función: Testear la funcionalidad de eliminación de archivos implementada en FileTreePanel.

Instrucciones:
1. Seleccionar este archivo en el TreeView
2. Presionar la tecla Suprimir
3. Confirmar la eliminación
4. Verificar que el archivo se elimina y se cierra si estaba abierto
